Local Storage

Saving data is important. Use JavaScript to save things to local storage.

localStorage.setItem(key, value) <- Use this to store a key value pair.

localStorage.getItem(key) <- Use this to retrieve a value from the key.

Mobile First Design

Designing for mobile first ensures a better user experience. Especially now that so many people use mobile devices.

Moving things that need to be clicked on to the bottom is good. On mobile, it's easier for them to be clicked on.

Stop using popup ads. Make your ads simple and unobtrusive. Put content first.
