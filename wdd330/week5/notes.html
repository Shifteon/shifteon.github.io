Exceptions

Exceptions return a value that let us handle an error. These are the kind of errors that will break the program. excpetions also produce a stack trace to help us find the source of the error.

Strict Mode

'use strict';

Strict mode makes JavaScript more strict. It throws more excpetions and keeps you from writing sloppy code. It is generally a smart idea to use strict mode so you write better code. To use strict mode just add 'use strict'; to the top of your JavaScript file, or at the top of a function if you want it to only apply to that function.

Feature Detection

Feature detection allows us to check if the browser supports a method before we use it.

                        
// make sure a feature exists before we use it
if (window.addEventListener) {
    window.addEventListener("load", function);
}

Throw, Try, Catch

We can throw our own exceptions using throw. Try let's us run a block of code, and if an exception is thrown we can catch the exception in the catch block to handle it gracefully.

                        
// throwing an error
throw new Error('Something went wrong!');

// try catch
function imaginarySquareRoot(number) {
    'use strict';
    try {
        return String(squareRoot(number));
    } catch(error) {
        return squareRoot(-number)+'i';
    }
}
