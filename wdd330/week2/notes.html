Data Structures

Arrays

How to initialize an array at creation

const avengers = ['Captain America', 'Iron Man', 'Thor', 'Hulk'];

JavaScript arrays can contain different types! This is very different from C++

Destructuring an array is really cool. This allows us to assign two values at the same time. For example:

const [x,y] = [1,2]

Set

A set is a data structure that does not allow duplicates.

const list = new Set(); // Create a set

This works because all non-primitive values, such as arrays and objects, are considered unique values, even if they contain the same values.

const arrays = new Set().add([1]).add([1]); // Set { [ 1 ], [ 1 ] }

Functions

            
// Normal function declaration
function foo() {
  console.log("hi");
}

// Assigning a function to a variable
const foo = function() {
  console.log("hi");
}

// Using the rest operator
function foo(...args) {
  for (arg of args) {
    console.log(arg);
  }
}

// Arrow function
const foo = (x, y) => {
  console.log("hi");
}

// Single parameter
const foo = x => x * x; // The statement returns because it's the only statement

A callback function is helpful when sorting numbers. We can tell the sort function how to sort with a callback.
