Transforms

Transforms allow us to translate, rotate, scale, and/or scale elements on the page.

Scale grows or shrinks an element based on x and y values. The elements center will always stay the same.

Rotate rotates an element on its point of origin by a specified angle.

Skew skews an element on the x and y axis.

The order of transforms matters. You may end up changing the point of origin and the a transform function might not work how you expect it.

                        
/* a translate function
   it allows us to move elements around
   based on their original position*/
transform: translate(45px, -45px);

/* the scale function. */
transform: scale(1.5, 0.25);

/* the rotate function */
transform: rotate(10deg);

/* the skew function */
transform: skew(15deg, 4deg);

/* you can change the orgin of the transform */
transform-origin: 10% 10%;

Transitions

Anything with a valid midpoint can be transitioned. Visibilty can also be transitioned.

                        
/* all of the transition properties */
.ad-ad2 h1 span {
    transition-property: transform; 
    transition-duration: 0.2s;  
    transition-timing-function: ease-out;
    transition-delay: 50ms;

    /* the shorthand */
    transition: transform 0.2s ease-out 50ms;

    /* transitioning on multiple elements */
    transition-property: transform, color; 
    transition-duration: 0.2s, 0.1s;  
    transition-timing-function: ease-out, linear;
    transition-delay: 50ms;

    /* with the shorthand */
    transition: transform 0.2s ease-out 50ms, color 0.1s linear 50ms;
}

Animations

Animations give us a lot more control over what happens than transitions by allowing us to set keyframes.

                        
/* keyframe rule for an animation */
@keyframes myAnimation { 
    /* put animation keyframes here */
}           

/* some filled out rules */
@keyframes moveRight { 
    /* from and to are 0% and 100% respectively */
    from {
    transform: translateX(-50%);
    }
    to {
    transform: translateX(50%);
    }
}

@keyframes appearDisappear { 
    0%, 100% {
    opacity: 0;
    }
    20%, 80% {
    opacity: 1;
    }
}

@keyframes bgMove {
    100% {
    background-position: 120% 0;
    }
}

/* applying an animation */
.verbose {
    animation-name: appearDisappear; 
    animation-duration: 300ms; 
    animation-timing-function: ease-in;
    animation-iteration-count: 1;
    animation-direction: alternate;
    animation-delay: 5s;
    animation-fill-mode: backwards;
    animation-play-state: running;
}

/* shorthand */
.concise {
    animation: 300ms ease-in alternate 5s backwards appearDisappear;
}

Canvas

Canvas lets us draw whatever we want with javascript.

Remember to give a canvas a border so it can be seen as it has no default styling.

                        
// draw a rectangle to a canvas
var canvas = document.getElementById("myCanvas"); 
var context = canvas.getContext("2d"); 
context.strokeStyle = "red";
context.fillStyle = "rgba(0, 0, 255, 0.5)";
context.fillRect(10, 10, 100, 100);   
context.strokeRect(10, 10, 100, 100);

SVG

SVG is similar to canvas in many ways. SVG is scalable vector graphics. This lets us resize the image without losing any of the quality. Creating our own complex image can be very intimidating, so using tools like photo editors and javascript libraries is a great idea.
