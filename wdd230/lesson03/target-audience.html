Target Audience

Audience Description

This website is built for runners in the Preston Idaho area. It's targeted more towards the younger generation as they use technology most often, but it will be simple enough to use that the older generations should find good use in it as well. The daily runner will find this site very useful for planning their runs ahead of time.

Personas
